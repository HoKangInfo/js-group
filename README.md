# js-group
a simple JavaScript Type or category of detection


### history ###
- 0.0.1    `isNumber`, `isInteger`, `isLength`, `isString`

### install ###
```
npm install js-group
```

### how to use it ###
```
var isString = require('js-group').isString;
isString(new String('this is a string'));
// ==> true
```
